Menene HTTP Server?

HTTP server shine wani tsari na musamman da ke bada damar musayar bayanai tsakanin mai amfani da server. Yakan yi aiki ne ta hanyar:

  • Karɓar buƙatun HTTP daga mai amfani
  • Aiwatar da buƙatun
  • Mayar da amsa zuwa ga mai amfani

Misalin Code Na Farko

            
              const http =require('node:http');

              const server = http.createServer();
              const PORT = 8000;

              server.on('request', (request, response) => {
                res.writeHead(200, {'Content-Type': 'text/plain'});
                res.end('Barka da zuwa server dina!');
              });

              server.listen(PORT, () => {
                console.log(`Server is running on port: ${PORT}`)
              });

Samar da Server
